 ### vault.raft_storage.follower.connection_status ((#vault-raft_storage-follower-connection_status))

Metric type | Value   | Description
----------- | ------- | -----------
gauge       | boolean | Indicates whether the follower nodes is connected

- A value of `1` on the gauge means that the follower is connected
- A value of `0` on the gauge means that the follower is disconnected
