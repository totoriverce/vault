# Vault License

Licenses and EULA enhancements have been introduced in Vault 1.8 release. Please refer to the [FAQs](/docs/enterprise/license/faqs) for common questions concerning these changes.

The [Install a HashiCorp Enterprise License](https://learn.hashicorp.com/tutorials/nomad/hashicorp-enterprise-license?in=vault/enterprise) tutorial provides the instruction to load your Vault license.
